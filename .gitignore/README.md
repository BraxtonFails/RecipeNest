# RecipeNest

RecipeNest is a Pinterest-style recipe sharing web application that allows users to discover, share, and save recipes in a visually organized grid layout. Users can browse meals, view ingredients and instructions, and favorite recipes for later, making meal planning and inspiration simple and accessible.

## Features
- Browse recipes in a Pinterest-style layout
- View ingredients and cooking instructions
- Save favorite recipes
- User-friendly and responsive design

## Tech Stack
- HTML, CSS, JavaScript
- MySQL
- VS Code
- GitHub Collaboration
